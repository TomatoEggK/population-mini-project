# population-mini-project
CIST final mini project about population growth and GDP per capita

## Thesis
In 2024, there is a negative relationship between population growth rate and GDP per capita across countries.

## Data Source
- World Bank Indicators API
- Population growth (annual %): SP.POP.GROW
- GDP per capita (current US$): NY.GDP.PCAP.CD

## Project Goal
This project tests whether countries with faster population growth tend to have lower GDP per capita.
